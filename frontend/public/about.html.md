# About Us - UNO With Friends

UNO With Friends is a personal, non-commercial fan project built to allow card game lovers to play UNO online with friends seamlessly.

---

## My Mission
UNO With Friends was created by a solo developer and card game enthusiast who wanted to build a seamless, zero-friction online platform for UNO lovers. My mission is to provide a fast, beautiful, and completely free gaming experience that connects friends across the globe without requiring sign-ups, downloads, or payment.

---

## The Technology
Built using modern web technologies, the frontend is powered by React, Vite, and Framer Motion for high-fidelity animations and responsive gaming. The backend uses Node.js, Express, and Socket.io to manage real-time, authoritative game states, card deals, action cards, and turn counters.

---

## Key Features
- **Private Multiplayer Lobbies**: Instantly share a 6-character code or link to play with friends online.
- **Intelligent CPU Bots**: Test your skills solo against smart bots that follow standard game strategies.
- **UNO Flip Mode**: Switch the deck dynamically between Light and Dark sides using the official Flip rules.
- **Cross-Platform Compatibility**: Play seamlessly on mobile phones, tablets, or desktop browsers.
